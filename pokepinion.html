PokéPinion Web App

Node.js, Express, MongoDB, HTML, CSS, JavaScript

Description

Web app that allows users to share and explore opinions on various Pokémon, providing a platform for engagement and community interaction via Pokémon data integration through PokéAPI and data persistence in MongoDB.

Details

  • Developed a web application using HTML, CSS, and JavaScript.
  • Designed and built a RESTful API with Node.js and Express for efficient data communication.
  • Integrated MongoDB for data storage, ensuring fast and reliable access to user data and Pokémon opinions.
  • Managed the project using Agile methodologies and regular Scrum meetings to ensure timely delivery and collaboration among team members.
  • Focused on design principles to create an intuitive and visually appealing interface to ensure a seamless user experience.
