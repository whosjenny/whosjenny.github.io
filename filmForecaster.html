Film Success Forecaster

Python, Machine Learning, Regression Analysis, Predictive Modeling, Scikit-learn, Data Preprocessing

Description

Machine Learning model that predicts the success of artificial movies based on various features such as budget, promotion, genre, and more. Dataset consisted of 540 movies. Final project for CMSC 320: Intro to Data Science.

Details

  • Developed a predictive model with > 79.6% accuracy.
  • Analyzed a dataset of 500+ films, including features such as genre, budget, cast, and release date, identifying budget as the most important feature.
  • Explored and preprocessed data from a variety of sources to identify key features influencing movie success.
  • Implemented various regression techniques to improve prediction accuracy.
  • Created visualizations to represent data trends and model performance.
  • Integrated the model into a web application for user-friendly interaction.
