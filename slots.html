Slot Machine Android App

Kotlin, App Development, Android Studio, Firebase, Google Ads, Event-Driven Programming

Description

Built an interactive slot machine game for Android with custom graphics, animations, persistent local and remote data storage, and sound effects, ensuring an enjoyable and addicting gaming experience.

Details

  • Developed a fully functional slot machine game for Android in Kotlin, allowing users to spin to win collectible stickers of varying rarity.
  • Allows users to create, save, and view one anothers accounts and earned stickers.
  • Implemented Model-View-Controller architecture consisting of multiple views to manage game logic, user interface, and data handling.
  • Utilized remote persistent data storage using Firebase for managing more voluminous game data as users create accounts.
  • Ensured fair and unpredictable game outcomes through the use of random number generation in the spirit of real slot machines.
  • Incorporated a fake ad pulled from Google Ads services to simulate advertising.
  • Used Android Studio for development and testing, ensuring compatibility across multiple devices.
  • Demoed completed project to class instructors, showcasing the app's features and functionality.
