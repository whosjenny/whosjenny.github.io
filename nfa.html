NFA to DFA Converter

OCaml, Graph Theory, Data Structures, Algorithms, Finite State Machines

Description

Implemented NFA and regular expression modules in OCaml, creating a regular expression interpreter and accompanying NFA-to-DFA converter, leveraging OCaml's type system and functional programming paradigms.

Details

  • Implemented functions to handle NFA operations including move, e_closure, and accept, ensuring accurate state transitions and string acceptance.
  • Built a parser for regular expressions, transforming them into NFAs and providing string representations using the regexp_to_nfa and regexp_to_string functions.
  • Employed the viz.native script for visualizing NFAs with Graphviz, aiding in debugging and validation.
  • Leveraged functional programming techniques to implement recursive functions and higher-order functions for operations like epsilon closure and fixpoint iteration.
